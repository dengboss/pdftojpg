文档批量转图片工具 - 打包说明

## 打包为exe独立文件

### 方法一：使用批处理文件（推荐）
1. 双击运行 `build_exe.bat`
2. 等待程序自动安装依赖并打包
3. 打包完成后，exe文件将生成在 `dist` 文件夹中
4. 文件名：`文档批量转图片工具.exe`

### 方法二：手动打包
1. 打开命令提示符（CMD）
2. 进入项目目录：
   ```
   cd /d 项目目录路径
   ```
3. 运行打包命令：
   ```
   python build_exe.py
   ```

### 打包后的exe文件特点
- ✅ 独立运行，无需安装Python
- ✅ 包含所有依赖库
- ✅ Windows系统直接运行
- ✅ 单文件，方便分发

### 系统要求
- Windows 7/8/10/11 (64位)
- 无需安装Python环境
- 建议4GB以上内存

### 文件结构
打包完成后，只需要以下文件即可运行：
```
dist/
└── 文档批量转图片工具.exe
```

### 分发方式
将 `文档批量转图片工具.exe` 文件发送给用户即可，用户无需安装任何其他软件。

### 注意事项
- 打包过程可能需要几分钟时间
- 生成的exe文件较大（约50-100MB），这是正常现象
- 建议在干净的Windows系统上测试exe文件

### 故障排除
如果打包失败：
1. 确保已安装Python 3.7或更高版本
2. 运行 `pip install -r requirements.txt` 安装依赖
3. 检查网络连接是否正常
4. 重新运行打包命令
